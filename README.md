# LSPosed Shortcut (Widget Edition)

A shortcut app to open LSPosed Manager directly from your home screen using a theme-adaptive App Widget, without cluttering your app drawer with launcher icons.

## How to Use

1. **Install the App:** Download and install the app from releases.
3. **Add Widget:** Search for the **LSPosed** widget in your widget drawer and add it to your home screen.
4. **Approve Root:** Grant root permission manually through your root manager. **If done already, go to app info and force stop once.**
5. **Done!** LSPosed Manager will launch. Future taps will launch LSPosed instantly.

## Features

- **No Launcher Icon:** The launcher activity icon is completely hidden from the app drawer to keep your interface clean.
- **Theme-Adaptive Widget:** A 1x1 App Widget that adapts to your home screen's Light/Dark mode and Material You dynamic color configurations (wallpaper color extraction).
- **Foreground Root Wakeup:** Tapping the widget launches a transparent activity, waking the app from Android's stopped state, requesting root permission in the foreground, and launching LSPosed Manager instantly.
